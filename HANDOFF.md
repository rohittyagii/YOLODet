# Handoff

Date: 2026-05-24

## How To Run
- Web YOLO: `python run.py` (opens browser)
- Backend only: `python run.py --mode backend`

## VS Code Tasks
- Run Project Web YOLO (Auto Setup)
- Run Project Backend Only (Auto Setup)
- Run Setup Wizard

## Notes
- Web UI expects backend on http://127.0.0.1:5000.
- YOLO models download on first use when using Ultralytics providers.
- If webcam access fails in browser, ensure permissions and close other camera apps.
